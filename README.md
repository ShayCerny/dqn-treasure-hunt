# DQN Treasure Hunt

A web application for training and visualizing a Deep Q-Network (DQN) agent solving custom maze environments in real time.

Build a maze, train a Double DQN agent with prioritized experience replay, and watch it learn via live charts and replay playback.

**Full source, docs, and local setup instructions: [github.com/ShayCerny/dqn-treasure-hunt](https://github.com/ShayCerny/dqn-treasure-hunt)**

> This Space runs on a free CPU tier with an ephemeral filesystem: mazes, trained models, and run history reset if the Space restarts or sleeps from inactivity.
